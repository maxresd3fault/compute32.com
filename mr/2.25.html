OVERVIEW

The Supermicro 6028R-E1CR24N is an excellent choice for a homelab storage server! This server packs 24 hot-swappable 3.5" drive bays into a compact 2U chassis, making it a fantastic option for high-capacity storage on a budget.

Right now, you can grab one for just $249.99 on The Server Store via eBay—this gets you the chassis along with two Intel Xeon E5-2620 CPUs without memory or drive trays. If you add 64GB of RAM and drive trays, the total cost comes to a still-reasonable $405.99 (with free shipping).

Pair this server with some used hard drives in RAID, and you’ve got yourself a massive, cost-effective storage solution with solid reliability. Out of the box, it supports RAID 0, 1, 5, 6, 10, 50, and 60, giving you plenty of flexibility.

One downside? It only takes 3.5" drive sleds, so you’ll need adapters if you want to use 2.5" SSDs. That said, the motherboard features two x16 PCIe slots, perfect for adding NVMe expansion cards if you need faster storage. There are also two rear 2.5" drive bays, which are ideal for installing an SSD or two as the boot drive.

While you can opt for higher-tier CPUs and more RAM, the dual E5-2620s (six cores each) should be plenty if you’re using this strictly as a storage server.

For the price, this Supermicro 24-bay is hard to beat! The eBay seller even includes a full one-year warranty.
