OVERVIEW

The QuantaGrid D51B is a great choice for an affordable compute server! This server has 10 hot-swappable 2.5" NVMe drive bays and comes configured with two Xeon E5-2650L processors, making it a great choice for multicore workloads.

Right now, you can grab one for just $341.99 with free shipping at The Server Store via eBay—this gets you the chassis along with the two Intel Xeon CPUs, 128GB of memory, and 10 blank drive trays. At the time of writing, you can also use the coupon code "SHOPANDSAVE" for an additional $15 off.

This machine would make a great first server for beginners as it's cheap, and comes with all the parts you need to get up and running (besides storage). A server like this is a great allrounder that is suitable for many different tasks.

One thing to keep in mind with this machine is that it uses 2.5" NVMe SSD's, even though they can visually look similar to a standard SATA SSD, they are not interchangeable. You will either have to find 2.5" NVMe SSD's for this machine, or install a more readily available storage solution via the PCIe slots. Server-grade 2.5" NVMe drives can be expensive, even used ones, so keep this in mind if you decide to purchase this machine.

For the price, this QuantaGrid is a great value, provided you can work out a storage solution that doesn't break the bank. The eBay seller also includes a full one-year warranty.
