OVERVIEW

The PowerEdge R630 is a great choice for an affordable compute server! This server has 8 hot-swappable 2.5" SATA/SAS hybrid drive bays and comes configured with two Xeon E5-2697 v3 processors for a total of 28 cores, making it a great choice for multicore workloads.

As of writing, you can get one on sale for just $295 with free shipping from TechMikeNY on eBay—this gets you the chassis along with the two Intel Xeon CPUs, 128GB of memory, and 8 blank drive trays.

This machine would make a great first server for beginners as it's cheap, and comes with all the parts you need to get up and running (besides storage). A server like this is a great allrounder that is suitable for many different tasks.

For the price, this Dell is a great value. I find Dell to make reliable and easily serviceable hardware. Furthermore Dell provides great support for their older products by continuing to host drivers, BIOS updates, and manuals. The eBay seller also includes a full one-year warranty.
