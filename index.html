Home

Compute32.com is a brand new website to find information about current deals and recommendations regarding surplus enterprise server equipment. Roughly every three to five years companies regularly upgrade server equipment, meaning there is an excess supply of surplus servers. These machines can typically be found for very cheap on retailers such as eBay because large companies are not in the used market for infrastructure, and the majority of home users are not buying rackmount hardware. Nevertheless, home users such as you and I can buy this equipment from various retailers and use it for personal use, this is known as "homelab".

Your homelab could be as simple as a Raspberry Pi ranging all the way to a server rack full of real enterprise grade machines. This website focuses on the latter and is meant to make obtaining and setting up this equipment easier for home users.

WHAT CAN I DO?

There are many projects that can be had with powerful surplus enterprise equipment. Some examples include but are not limited to:

  • Host a local AI model
  • Host a website
  • Set up a storage server for your home
  • Set up a "rendering farm" for programs like Blender
  • Host various video game servers
  • Host a media server for your home

There are many ways to find a new use for old servers, even if you have no need for these services much can be learned from tinkering. In addition, giving these devices a second life keeps them out of landfills, upcycling technology is one of the primary goals of acquiring this kind of equipment.
